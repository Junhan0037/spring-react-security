# spring-react-security

본 프로젝트는 Spring Security, React 를 사용하여 기본적인 회원 관리 예제 프로젝트 입니다.

### Start

설치 및 개발서버 실행

```sh
# 1. 설치
$ npm install

# 2. Java 실행
$ ./gradlew clean bootRun

# 3. 개발서버 실행
$ npm start
```
